04

Generated — connective tissue

Hero texture and icon set, generated to hold one style: pine + clay only, thin consistent stroke weight, the same diagonal-line motif as the logo mark. A node-link pattern was chosen deliberately — it echoes the trees, graphs, and hash tables that are literally what the Google Drive project is built from, so the texture is on-subject rather than decorative.

05

Rejected — and why

Rejected

First hero pass was a dense circuit-board trace pattern — the generic "developer portfolio" texture. Rejected because it's a tech cliché that says generic tech, not this work: it doesn't reference anything Umar actually built (no swap, no ledger, no tree), and the busy right angles fight the calm kit mood instead of sitting quietly behind it.

06

Where real beat AI

  • Real Three project screenshots — proof has to be the actual repo, not a plausible-looking mockup of features that may not render the same way.
  • Real About photo — the person on the page has to be the actual person.
  • Generated Hero texture + 3 icons — pure connective tissue with no factual claim, generated once and held to one style so the page doesn't feel like a pile of unrelated assets.
